# Memory Bank

Persistent context for AI sessions. **Read all files at the start of every task.**

## Core Files

| File | Purpose |
|------|---------|
| projectbrief.md | Scope, requirements, success criteria |
| productContext.md | Why, problems, how it works |
| systemPatterns.md | Architecture, decisions, component flow |
| techContext.md | Stack, MCP servers, constraints |
| activeContext.md | Current focus, next steps |
| progress.md | What works, what's left, known issues |
| journal.md | Learned patterns, preferences (update when discovering) |

## Update Triggers

- After implementing significant changes
- When user says "update memory bank"
- When context needs clarification

## Priority

Memory Bank supersedes other rules, skills, and agents when there is conflict.
